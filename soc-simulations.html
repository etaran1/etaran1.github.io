Phishing Email → Credential Attempt (Triage + Escalation)

Role Simulation: SOC Analyst (Tier 1–2)
SOC / Triage
Scenario

User reports a suspicious email; shortly after, identity provider logs show unusual sign-in attempts from a new location/device.

Analysis & Decisions
  • Validated user report and extracted indicators (sender domain, URLs, attachment hashes).
  • Checked IdP authentication logs for impossible travel, new device, MFA failures, and token activity.
  • Determined severity and escalated based on confirmed credential-use signals.
  • Recommended containment: password reset, session/token revocation, and conditional access tightening.
Escalation rationale: Confirmed authentication anomalies + user-reported phishing indicates active credential compromise risk.
Outcome
  • Phish blocked across tenant; malicious domain and URLs added to deny lists.
  • Account secured with forced reset and session revocation; monitoring placed on related accounts.
Artifacts Produced
Triage Notes IoC List (domains/URLs/hashes) Escalation Summary Containment Checklist
Reflection
  • Token/session revocation is often faster than endpoint action for immediate containment.
  • Conditional access + risky sign-in policies reduce repeat incidents.

Endpoint Malware Alert → Host Containment + Evidence Collection

Role Simulation: SOC Analyst (Tier 1–2)
SOC / IR
Scenario

EDR triggers on suspicious process behavior (possible credential dumping or persistence). Host shows unusual outbound connections.

Analysis & Decisions
  • Confirmed alert fidelity (process tree, command line, parent/child relationships).
  • Reviewed network telemetry for C2 patterns, DNS anomalies, and rare destinations.
  • Decided containment scope (isolate host vs. block indicators) based on impact and confidence.
  • Collected evidence for IR: timeline notes, key logs, hashes, and affected user context.
Containment choice: Isolate the host when behavior indicates credential access or persistence—limit blast radius first, then investigate deeper.
Outcome
  • Host isolated and indicators blocked; IR case opened with evidence bundle.
  • Follow-up monitoring placed on related endpoints and user accounts.
Artifacts Produced
Incident Timeline EDR Evidence Summary Network IoCs Escalation + Handoff Notes
Reflection
  • Consistent handoff notes reduce time-to-containment for Tier 2/IR teams.
  • Process-tree context beats single-event alerts for confidence scoring.
